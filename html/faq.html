Can I find all member state payments here?

We are trying to make payments from as much countries as possible available for search, but there are technical limitations, so some countries are missing. The country filter on the page header gives you an overview of countries currently available.


  Can I trust the data?

Since this is an additional link in the chain to the actual payment data, there is the possibility of additional mistakes, though we have implemented various checks when crawling the data.

If you have found subsidy payments you want to base some research/journalistic work on, always make a cross-check on the corresponding EU member state subsidy database.

  How can I give you some credit?

If you reference payments in your publication, always use the corresponding EU member state subsidy database as your primary source. We are happy though, if you reference OpenFarmsubsidies as your tool for investigation.


  I receive subsidies and don't want my name appear.

OpenFarmsubsidies is not a database but a search engine referencing subsidy data found in the EU member state databases. Please contact the responsible publication authority of your country if you have questions regarding the publication of your subsidy payment.
